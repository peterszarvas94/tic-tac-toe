# Tic tac toe but bigger

## Prerequisites
- make sure [node.js](https://nodejs.org/) is installed

## Running the app

Open a terminal, and run:

```
npm install
npm run build
npm run start
```

Go to `localhost:3000` or any other address the terminal shows

The username and password are both `admin`
